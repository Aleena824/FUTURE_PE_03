# FUTURE_PE_03

## Task 3 — AI SEO Blog & Content Cluster Generator

### Overview

Most businesses don't fail at SEO (Search Engine Optimisation) because their content is bad, they fail because they post randomly with no structure. Real SEO agencies build content clusters: one strong pillar article supported by smaller blogs, all internally linked, all targeting related keywords. This task replicates that system using structured prompts.

### Business Chosen

**Apex Digital** — Digital Marketing Agency, Dubai, UAE

A fictional Dubai-based digital marketing agency. Chosen for its competitive niche, clear local SEO opportunities, and relevance to a professional audience.

 (Note: Apex Digital is a fictional business created for educational/portfolio purposes.)

### Prompt Logic

The framework uses a **5-layer architecture**:

```
Layer 1: Business & SEO Context    --> Business type, location, keywords, audience
Layer 2: Keyword Strategy          --> Primary + secondary keywords, search intent mapping
Layer 3: Pillar Blog Generation    --> Long-form, authority-building main article (~1,400 words)
Layer 4: Supporting Blog Cluster   --> 4 smaller blogs that link back to the pillar
Layer 5: Local SEO Adaptation      --> City + service keyword integration
```
